# Notality — Website

Marketing site for [Notality](https://notality.app).

Built with plain HTML, CSS, and JavaScript — no build step required.

## Structure

```
Notality_Domain/
├── index.html          # Landing page
├── pages/
│   └── privacy.html    # Privacy policy
├── css/
│   ├── reset.css       # CSS reset
│   └── styles.css      # Main styles
├── js/
│   └── main.js         # JavaScript
└── assets/
    ├── images/         # Screenshots, photos
    └── icons/          # Favicon and app icons
```

## Development

Open `index.html` directly in a browser, or use the VS Code Live Server extension for auto-reload.
